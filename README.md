# Projet M-Motors Digital

Bienvenue dans mon dépôt pour le projet M-Motors Digital. L'idée ici est simple: répondre à un besoin de mobilité moderne en rendant l'accès à l'automobile fluide sans prise de tête.

## Pourquoi ce projet?

M-Motors existe depuis 1987. C'est un spécialiste de l'occasion qui fait partie du top 10 national pour sa satisfaction client. Mais aujourd'hui, le défi est ailleurs : il faut effacer la limite entre l'achat classique et la location longue durée (LLD).

M-Motors Digital, c'est le nom de ma solution pour permettre aux clients de gérer tout ça sereinement, directement depuis leur canapé.

## Ce que j'ai construit

Pour mon Bachelor, j'ai choisi de me concentrer sur trois points essentiels:

1. Un catalogue clair qui ne mélange pas l'achat et la location de voiture.
2. Un espace sécurisé pour que les cliens déposent leurs pièces justificatives (Permis de conduire, Carte d'identité).
3. Un back-office simple pour que les équipes de M-Motors puissent valider leur dossier facilement.

## Mes choix techniques

J'ai misé sur des outils solides pour que le service soit stable.

- Pour l'éxperience utilisateur : HTML5 et CSS3
- Pour le moteur de l'application : Java JEE. C'est puissant et c'est ce qu'il y a de plus sûr pour gérer des données sensibles.
- Pour la base de données : MySQL.
- Pour le suivi du code et la sécurité du projet : Git et GitHub.
